CSS Snippet Cheat Sheet

Ever have trouble remembering your favorite CSS code? Give it a permanent home and add it to this page! Select any snippet below to automatically select all of the code for you to copy.

Flexbox Row

Use these three properties to make a flexbox row layout.

.row {
display: flex;
flex-direction: row;
flex-wrap: wrap;
}

Flexbox Column

Use these three properties to make a flexbox row layout.

.row {
display: flex;
flex-direction: column;
}

CSS Grid Layout

Build a 12 column layout using CSS grid.

.grid {
display: grid;
width: 100%;
grid-template-columns: repeat(12, 1fr);
}

Linear Gradients

Use these properties to create a more dynamic background color.

.linear-gradient-background {
background-image: linear-gradient(#008080, #71c4c4);
}

Box Transition Glow

Use transition and box shadow to glow on hover.

.code-card .card-header {
border-radius: 8px;
transition: all 0.5s ease-in-out;
}

.code-card:hover .card-header { box-shadow: inset 0px 0px 8px #008080, 0 0 15px #71c4c4; }
